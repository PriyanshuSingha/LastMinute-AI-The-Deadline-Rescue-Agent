# LastMinute AI – The Deadline Rescue Agent

## Problem Statement

### The Last-Minute Life Saver

Students, professionals, and entrepreneurs frequently miss deadlines, assignments, meetings, bill payments, interviews, and important commitments. Existing productivity tools often rely on passive reminders that are easy to ignore and do little to help users actually complete their tasks.

## Solution Overview

LastMinute AI is an AI-powered productivity companion built using Google's Gemini AI. The application proactively assists users in planning, prioritizing, and completing tasks before deadlines are missed.

Unlike traditional reminder applications, LastMinute AI analyzes deadlines, predicts risks, generates rescue plans, and provides actionable recommendations to help users complete their work effectively.

## Key Features

* AI-Powered Task Prioritization
* Smart Daily Planner
* Deadline Risk Prediction
* Emergency Rescue Mode
* Productivity Analytics
* Goal and Habit Tracking
* Calendar Management
* Gemini-Powered Productivity Coach
* Context-Aware Recommendations

## System Workflow

1. User creates tasks and deadlines.
2. Gemini analyzes urgency and effort.
3. AI generates priority scores.
4. Smart planner creates schedules.
5. Risk predictor identifies potential missed deadlines.
6. Rescue Mode generates recovery plans.
7. Productivity coach provides recommendations.
